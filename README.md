# BuzzOnCampus

Live 3D campus map platform for university students. Post events, volunteer opportunities, and help requests pinned to real campus locations.

**Live:** https://buzzoncampus-f9257.web.app

---

## Stack

- **Frontend:** React 18 + Vite + TypeScript, Mapbox GL JS, React Three Fiber, Tailwind CSS, Zustand
- **Backend:** Firebase — Auth, Firestore, Cloud Functions, Hosting

## Features

- University-themed UI per campus (YSU, Kent State, OSU)
- Real-time pin map with 3D models per pin type
- OTP-verified `.edu` email signup
- Buzz Points economy — earn points by joining events and volunteering
- Volunteer hours tracking with admin approval flow
- Admin panel scoped per university
